ToDo List App

Descrição: Aplicação para gerenciamento de projetos e tarefas diárias.

Objetivo: Auxiliar na organização de tarefas dos projetos.

Entidades: 
- Projetos:
	- Nome
	- Descrição
	- Data de criação
	- Data de atualização

- Tarefas:
	- Nome
	- Descrição
	- Status
	- Observações
	- Prazo
	- Data de criação
	- Data de atualização

Requisitos: 
- Criar, alterar e deletar o projeto. 
- Criar, alterar e deletar a tarefa.

Regras de negócio:
- O sistema não contará com um sistema de login.
- Não haverá o conceito de usuário.
- Toda tarefa deve pertencer a um projeto.

Tecnologias utilizadas:
- Java
- MySQL
